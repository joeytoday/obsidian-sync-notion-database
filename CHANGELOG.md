## [1.1.0] - 2026-03-05

### 新增
- **同步中心 (Sync Center)**: 全新的可视化同步界面
  - 支持预览即将同步的文件列表
  - 可选择性勾选需要同步的文件
  - 显示新增、更新、未变更的文件统计
  - 支持全选/取消全选/仅选新增/仅选更新
  - 新增文件支持预览，更新文件支持差异对比
- **Relation 属性解析**: 自动获取关联页面的标题（而非仅显示 ID）
- **增强的属性类型支持**:
  - Formula: 支持 string/number/boolean/date 多种返回类型
  - Rollup: 支持 array/number/date 类型，自动提取数组中的值
  - Files: 支持 external 和 file 类型，提取 URL

### 改进
- **YAML Frontmatter 格式化**:
  - ISO 时间格式自动转换为 `YYYY-MM-DD HH:mm`
  - 改进多行文本的 YAML 块标量处理
  - 智能判断 YAML 引号需求，URL 等无需引号
  - 正确处理数字、日期和特殊字符
- **模板系统**: 支持模板 frontmatter 与 Notion 数据智能合并
  - 列表类型属性（如 tags）支持模板默认值与 Notion 数据合并
- **错误处理**: 增强错误处理和日志记录，添加分页获取数据的日志输出
- **稳定性**: 防止重复处理同一页面

## [1.0.1] - 2026-03-02

### 新增
- **文件选择弹窗**: 支持用户自定义同步文件
  - 支持文件列表构建及同步规则过滤
  - 支持勾选、全选、取消全选和覆盖选项
  - 展示文件状态（已存在/新建）及覆盖勾选框
- **同步结果统计**: 分别统计新建、更新、未变更及跳过数量
- **测试连接按钮**: 在设置页新增按钮，便于验证 Notion API 接口可用性

### 改进
- 优化同步出错提示，增强异常捕获与用户反馈

## [1.0.0] - 2026-02-24

### 新增
- **初始版本发布**: 完成 Notion 数据库同步插件的初始实现
- **核心同步功能**: 将 Notion 数据库记录同步到 Obsidian 笔记
- **属性映射**: 支持自定义 Notion 属性与 Obsidian Frontmatter 的映射
- **同步规则**: 支持基于属性条件的同步过滤
- **文件模板**: 支持自定义文件模板，使用占位符动态生成内容
- **本地模板文件**: 支持使用本地文件作为同步模板
- **自定义 NotionClient**: 基于 Obsidian requestUrl 实现，避免 CORS 问题

### 技术细节
- 使用 Obsidian 原生 API 进行文件操作
- 支持分页获取 Notion 数据库记录
- 自动处理文件名冲突
- 基于文件修改时间判断是否需要更新
